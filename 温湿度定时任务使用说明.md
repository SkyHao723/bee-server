# 温湿度定时任务使用说明

## 功能说明
定时向 `beehive_temperature` 表插入温湿度假数据，用于测试。

- **apiary_id**: 固定为 1
- **beehive_id**: 固定为 1
- **温度范围**: -10 到 60 度
- **湿度范围**: 30 到 100
- **执行频率**: 每 1 分钟执行一次

## 如何关闭定时任务

### 方式1：注释 @Scheduled 注解（推荐）

打开文件：`ruoyi-quartz/src/main/java/com/ruoyi/quartz/task/BeehiveTemperatureTask.java`

```java
// 注释掉这一行即可关闭
// @Scheduled(cron = "0 */1 * * * ?")
public void generateTemperatureData()
```

需要重新启用时，取消注释即可。

### 方式2：修改 cron 表达式

将 cron 表达式改为无效值：

```java
@Scheduled(cron = "-")
public void generateTemperatureData()
```

## 如何修改执行频率

修改 `@Scheduled` 注解中的 cron 表达式：

```java
// 每 1 分钟执行一次
@Scheduled(cron = "0 */1 * * * ?")

// 每 5 分钟执行一次
@Scheduled(cron = "0 */5 * * * ?")

// 每 10 分钟执行一次
@Scheduled(cron = "0 */10 * * * ?")

// 每小时执行一次
@Scheduled(cron = "0 0 * * * ?")

// 每天凌晨执行一次
@Scheduled(cron = "0 0 0 * * ?")
```

## 文件位置
